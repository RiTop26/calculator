# calculator
For this project, we are going to make on-screen calculator using JavaScript, HTML, and CSS.

Set up functions for: add, substract, multiply, and divide.

Create three variables for a number, an operator, and another number.

Create a new function operate that takes an orperator and two numbers and calls one of the above functions on the numbers.

Create a basic HTML calculator with buttons for each digit and operator.

Create the functions that update one of the number variables when the calculator's digit buttons are clicked
